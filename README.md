# Speech-Emotion-Recognition
# Speech Emotion Recognition using Hand-Crafted Audio Features

## Project Overview

This project focuses on developing a **Speech Emotion Recognition (SER)** system that identifies human emotions from speech signals using carefully selected hand-crafted audio features and deep learning models.

## Datasets Used (English)

The system is trained using four widely used English speech emotion datasets:

* **RAVDESS**
* **TESS**
* **SAVEE**
* **CREMA-D**

## Features Extracted

Four acoustic features are extracted from the speech signals:

* **MFCC (Mel-Frequency Cepstral Coefficients)**
* **Zero Crossing Rate (ZCR)**
* **Root Mean Square Energy (RMSE)**
* **Chroma STFT**

## Models Implemented

Three machine learning models are developed and evaluated:

* **CNN (Convolutional Neural Network)**
* **BiLSTM (Bidirectional Long Short-Term Memory)**
* **Ensemble Model**
